# Slio
## 개요
### 프로젝트 이름

Slio - 슬리오
Portfolio의 folio는 종이를 뜻하는데 이 folio에 넘기다 라는 Slide를 섞어 만든 이름이다


## 타겟층

포트폴리오를 공유하고 열람하고 싶은 사람 누구나

## 핵심 기능

1. 포트폴리오 탐색
    - 포트폴리오 슬라이드
    - 거리 및 취향별 포트폴리오 정렬
2. 포트폴리오 공유
    - 다양한 형식으로 포트폴리오 올리기 (사진,영상,글,음악,PDF)
3. 포트폴리오 관리
    - 포트폴리오 잔디

## 지원 플랫폼

android / Ios
